# Tahir AI Writer

  Full-stack AI-powered WordPress content management platform.

  ## Features
  - AI article generation via LongCat AI (LongCat-Flash-Chat / LongCat-Flash-Thinking-2601)
  - WordPress site management with OAuth application passwords  
  - Auto-publishing or draft saving
  - Pollinations AI featured images (uploaded directly to WP media library)
  - Auto-generated SEO: meta description, tags, category, keyword-based slug
  - Scheduled publishing with background queue (5 concurrent)
  - Article history and user management (admin/author roles)

  ## Tech Stack
  - **Frontend**: React + Vite + TypeScript + Tailwind CSS + shadcn/ui
  - **Backend**: Express + TypeScript + Drizzle ORM + PostgreSQL
  - **AI**: LongCat API (api.longcat.chat)
  - **Images**: Pollinations.AI
